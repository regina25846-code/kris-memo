설치 및 라이선스 인증

설치

  1. K-Memo Setup x.x.x.exe 다운로드 후 실행
  2. Windows 보안 경고 → "추가 정보" → "그래도 실행" 클릭
  3. 설치 완료 후 K-Memo 실행
포터블 버전(K-Memo-x.x.x-portable.exe)은 설치 없이 바로 실행 가능합니다.

라이선스 인증

  1. 첫 실행 시 라이선스 키 입력 화면이 표시됩니다
  2. 발급받은 키(KMEMO-XXXX-XXXX-XXXX 형식) 입력
  3. 인증 완료 후 앱이 시작됩니다
라이선스 키는 PC 1대에만 등록됩니다. 다른 PC에서 사용하려면 별도 키가 필요합니다.
2

화면 구성

화면 오른쪽(또는 왼쪽)에 사이드바가 고정되어 표시됩니다.

  • 상단 버튼: 새 메모(＋), 메모 관리(📋), 백업(🔄), 설정(⚙)
  • 탭 바: 📝 메모 · 🕐 기본 · 📅 스케줄 · ⭐ 즐겨찾기
  • 메모 목록: 폴더별로 분류된 메모 목록
사이드바 위치(좌/우), 테마는 설정에서 변경할 수 있습니다.
3

메모 작성 및 관리

새 메모 작성

  • 상단 ＋ 버튼 클릭 → 화면에 새 메모 창 표시
  • 제목과 내용 입력 후 자동 저장
  • 메모 창은 항상 최상위에 표시되어 작업 중에도 확인 가능

텍스트 서식 (굵게)

  • 메모 안에서 글자를 드래그로 선택한 뒤 하단 B 버튼 클릭 → 굵게 표시
  • 굵게 표시는 메모를 닫았다 다시 열어도 그대로 유지됨
  • 이미 진한 폰트(예: KoPubWorld Bold)를 쓰는 중이어도 B 버튼을 누르면 더 확실하게 두꺼워 보임

메모 우클릭 메뉴

메모 목록에서 항목을 우클릭하면:

  • 열기 — 메모 창 열기
  • 즐겨찾기 추가/해제 — 별표(⭐) 표시
  • 삭제 — 메모 삭제

폴더 분류

  • 메모 관리(📋)에서 폴더 추가/편집/이동 가능
  • 폴더에 비밀번호 설정 가능 — 비밀번호 설정은 환경설정 → 보안에서 이메일 복구 설정을 한 후에 가능
4

기본 탭 — 오늘 할 일

🕐 탭에서 오늘 날짜/요일과 함께 간단한 메모를 작성할 수 있습니다.

  • 날짜/요일은 매일 자동으로 오늘 날짜로 표시
  • 메모 내용은 직접 지우기 전까지 유지
  • 당일 스케줄이 있으면 상단에 함께 표시
내용은 날짜가 바뀌어도 사라지지 않아요. 직접 지워야 초기화됩니다.

할 것 영역 색상 지정

  • 우클릭 → 색상 지정으로 배경색/글자색 직접 설정 가능
  • 색상 정사각형 클릭 → OS 색상표에서 시각적으로 색상 선택
  • hex 코드 직접 입력 가능 (예: #FF8800)
  • 스포이드 버튼으로 모니터 전체 화면에서 색상 추출 지원
  • "자동" 버튼으로 배경색에 맞는 글자색 자동 설정
5

스케줄 탭

  • + 추가 버튼으로 날짜/시간/제목 입력
  • 알람 설정 시 해당 시간에 알림 표시
  • 하루 종일 일정은 한국시간 오전 9시를 기준으로 알림 — 예: "10분 전" 설정 시 오전 8:50에 알림
  • D-Day 설정 가능 (중요 날짜 카운트다운)
  • 완료된 일정은 취소선 처리, 목록 하단으로 이동
  • 기본 탭에서 오늘 스케줄이 자동 표시
  • 달력에서 오늘 날짜는 동그라미로, 일정 있는 날은 밑줄로 표시
v1.2.1 — 하루 종일 일정의 알림 기준 시간이 오전 9시(KST)로 통일되었습니다.
6

즐겨찾기 탭

  • 메모 우클릭 → 즐겨찾기 추가 시 ⭐ 탭에 모여서 표시
  • 즐겨찾기 탭에서도 우클릭 → 즐겨찾기 해제 / 열기 / 삭제 가능
즐겨찾기 탭에서도 우클릭 메뉴가 지원됩니다. (v1.0.0 추가)
7

폴더 및 메모 관리

상단 📋 버튼으로 메모 관리 화면 진입:

  • 여러 메모 선택 후 일괄 삭제 / 폴더 이동
  • + 폴더 버튼으로 새 폴더 추가
  • 폴더 편집으로 이름 변경 / 비밀번호 설정
  • 하위 폴더 — 폴더 우클릭 → "하위 폴더 추가"로 계층 구조 구성 가능
v1.2.1 — 하위 폴더 기능이 추가되었습니다. 폴더를 우클릭하면 하위 폴더를 추가할 수 있습니다.
8

설정

시작 / 실행

  • 시작 프로그램 — Windows 부팅 시 자동 실행
  • 시작 시 사이드바 표시 — 앱 시작 시 사이드바 자동 열기

자동 백업

  • 매일 설정한 시간에 자동으로 메모 데이터 백업
  • 최근 30일치 보관

단축키

  • 사이드바 표시/숨기기 단축키 직접 설정 가능

테마

  • Fluent · VSCode · Paper · Frost 4종 테마 선택 — 클릭 즉시 실시간 미리보기

폰트

  • 메모장 폰트 — Inter / Pretendard / KoPubWorld Dotum Light·Medium·Bold 중 선택
  • 사이드바(시스템) 폰트 — 사이드바 전체 폰트 별도 선택
  • 폰트 크기 — 기본 / 크게+1 / 크게+2 / 크게+3 단계별 선택
  • 드롭다운 변경 시 저장 전에도 실시간으로 미리보기 적용
  • 메모 창 안에서 우클릭 → 폰트 크기 변경 가능 (전체 메모에 즉시 적용)
v1.2.2 — 메모장·사이드바 폰트를 별도로 설정할 수 있으며, 우클릭으로도 폰트 크기를 바꿀 수 있습니다.

사이드바

  • 환경설정 → 사이드바 탭에서 위치(좌/우), 자동 숨기기, 항상 위 등 통합 관리
  • 항상 위 옵션: 사이드바가 다른 창 위에 항상 표시
  • 자동 숨기기 옵션: 포커스를 잃으면 사이드바 자동 숨김
9

백업 / 복원

  • 상단 🔄 버튼 → 수동 백업 / 복원
  • 백업 파일 열기로 저장 위치 확인
  • 복원 시 현재 데이터가 백업 시점으로 되돌아감
복원하면 현재 작성한 내용이 사라집니다. 중요한 내용은 먼저 백업하세요.
10

보안 — 폴더 잠금 & 이메일 복구

폴더 잠금

  • 폴더 잠금(비밀번호 설정)은 환경설정 → 보안에서 이메일 복구 설정을 한 후에 사용 가능
  • 잠긴 폴더는 비밀번호 입력 전까지 내용 숨김

이메일 복구

  • 비밀번호를 잊었을 때 이메일로 임시 코드 발송
  • 설정 → 보안 → 이메일 복구 설정에서 Gmail 등록 필요
  • Gmail 앱 비밀번호 생성 방법은 화면 내 안내 참고
11

업데이트

설치형 (자동 업데이트)

  • 새 버전 출시 시 앱 내 배너 자동 표시
  • "클릭하여 재시작" → 자동 설치 완료

포터블 (수동 업데이트)

  • 새 버전 있으면 "새 버전 vX.X.X — 클릭하여 다운로드" 배너 표시
  • 배너 클릭 → 다운로드 페이지 이동
